# 비디오 편집 브레인 팩 — 릴스/쇼츠 노하우

## 9:16 릴스 황금 구조

### 15초 릴스 (가장 안전)

```
0~2초    후킹 (시각 + 자막)
2~6초    문제 제기
6~10초   사용 장면 (메인)
10~13초  결과 (변화 보여줌)
13~15초  CTA
```

### 30초 릴스

```
0~3초    후킹
3~10초   문제 + 자극
10~22초  사용 장면 + 변화
22~27초  리뷰/증언 컷
27~30초  CTA + 브랜드
```

## 컷 길이 룰

- **2초 이상 같은 화면 금지** (시청자 이탈)
- 평균 컷 길이 **1.0~1.5초**가 최적
- 클로즈업/풀샷 교차로 다이내믹 유지

## 자막 디자인

- 폰트: **Malgun Gothic Bold** 또는 **Pretendard Black**
- 크기: 화면 높이의 **6~8%**
- 위치: **중앙 또는 하단 1/3 지점**
- 외곽선: 검정 4px (가독성)
- 색상: 흰색 + 핵심 단어만 노란색/빨강
- 한 컷에 **한 문장만** (분할 자막 X)

좋은 예: `사냥 본능이 안 켜져요?` (한 줄)
나쁜 예: `우리집 고양이가 사냥본능이 잘 안 켜지는데 어떻게 해야 할까요?` (긴 줄)

## BGM 선택 룰

| 영상 종류 | BGM 유형 | BPM |
|---|---|---|
| 사냥본능/활동 | 업비트 EDM/하우스 | 110~130 |
| 잔잔/감성 | 어쿠스틱 | 70~90 |
| 코믹/일상 | 로파이/재즈 | 80~100 |
| 광고 후킹 강함 | 임팩트 효과음 + 짧은 비트 | 변동 |

**라이선스 필수** — YouTube 오디오 라이브러리, Epidemic Sound, Artlist 같은 곳에서만.
저작권 BGM 사용 시 — 광고 차단, 수익화 불가, 법적 책임.

## ffmpeg 한 줄 명령 모음

### mp4 합치기 (concat)
```bash
ffmpeg -f concat -safe 0 -i inputs.txt -c:v libx264 -r 30 out.mp4
```

### 9:16 비율로 자르기
```bash
ffmpeg -i input.mp4 -vf "scale=1080:1920:force_original_aspect_ratio=increase,crop=1080:1920" out.mp4
```

### 자막 박기 (SRT)
```bash
ffmpeg -i input.mp4 -vf "subtitles=sub.srt:force_style='Fontsize=48,PrimaryColour=&Hffffff,OutlineColour=&H000000,BorderStyle=3'" out.mp4
```

### BGM 합성 (원본 음성 + BGM 40%)
```bash
ffmpeg -i video.mp4 -i bgm.mp3 -filter_complex "[1:a]volume=0.4[bgm];[0:a][bgm]amix=inputs=2:duration=first" out.mp4
```

## Premiere Pro 자동화 핵심

- **FCPXML 임포트** — `File > Import` → 시퀀스 자동 생성
- **ExtendScript .jsx** — `File > Scripts > Run Script File...` → bin 자동
- **저작권 안전:** 새 bin/새 시퀀스만 만들고 기존 프로젝트 절대 안 건드림

## 썸네일 디자인 4룰

릴스/쇼츠 썸네일은 **3초 안에 클릭 결정**:

1. **얼굴/표정** (고양이 클로즈업) — 시선 즉시 잡힘
2. **굵은 키워드 2단어** — "안 노는 고양이"
3. **대비색** — 노랑+검정, 분홍+검정
4. **숫자/기호** — "3가지", "↑", "?"

❌ 피하기:
- 풍경 사진
- 작은 글씨
- 5단어 이상
