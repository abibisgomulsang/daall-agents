# 고양이 사업 브레인 팩 — 도메인 지식

## 사장님 회사 (아비비의고물상) 기본 정보

- **법인:** 주식회사 다올에프엔에스
- **스토어:** 아비비의고물상 (ABIBI'S GOMULSANG)
- **플랫폼:** 네이버 스마트스토어
- **상품군:** 수제 고양이 장난감 (낚싯대, 깃털 리필, 곤충 모양)
- **브랜드 마스코트:** 3마리 고양이 → ABIBI (아치 + 일비 + 단비)
  - **아치** — 턱시도(검정+흰색), 스크래쳐 모델 담당
  - **일비** — 주황+흰색, 당당한 성격
  - **단비** — 고등어무늬

## 고양이 성향 5분류 (집사가 자기 고양이 분류하게 만드는 마케팅 도구)

| 성향 | 특징 | 추천 장난감 |
|---|---|---|
| **사냥꾼형** | 활동량 많음, 점프·추격 좋아함 | 고스틱(긴 낚싯대), 깃털 |
| **관찰자형** | 멀리서 지켜봄, 신중함 | 천천히 흔드는 낚싯대, 부드러운 인형 |
| **수다쟁이형** | 야옹야옹 많음, 집사 따라다님 | 같이 노는 깃털, 인터랙티브 |
| **잠꾸러기형** | 활동량 적음, 자극에 무관심 | 짧고 강한 자극 (반짝이는 리본) |
| **새끼고양이형** | 1살 미만, 호기심 폭발 | 작고 안전한 곤충 모양, 부드러운 끈 |

→ 상세페이지에 "우리 집 고양이는 어떤 성향?" 퀴즈 유도하면 전환율 ↑

## 고양이 장난감 시장 키워드 (네이버 검색량 기준)

상위 키워드:
- 고양이 낚싯대 (월 4만 검색)
- 고양이 장난감 (월 12만)
- 깃털 낚싯대
- 사냥 놀이
- 고양이 캣닢

블루오션 키워드 (경쟁 적음):
- 고양이 성향별 장난감
- 활동량 많은 고양이
- 사냥본능 자극
- 리필 가능한 낚싯대
- 수제 고양이 장난감 (사장님 강점)

## 고양이 장난감 광고 법적 주의 사항

❌ 금지:
- "스트레스 해소" (의학적 효능)
- "건강에 좋은" (의학적 표현)
- "100% 안전" (절대적 표현)
- "고양이가 무조건 좋아함" (단정 표현)

✅ OK:
- "사냥 본능 자극"
- "활동량 늘리기"
- "놀이 시간 늘릴 수 있어요"
- "성향에 맞춰"

## 리뷰 분석 시 보는 키워드 (긍정/부정)

**긍정 키워드 (강조 활용):**
- "잘 놀아요", "오래 가지고 놀아요", "튼튼해요", "리필 있어서 좋아요", "고양이가 좋아함"

**부정 키워드 (개선 우선):**
- "금방 망가져요", "깃털이 빠져요", "끈이 끊어져요", "포장이 약해요", "배송이 늦어요"

→ 부정 키워드 톱 3을 매주 점검해서 검수 AI가 상세페이지 보완 제안

## 가격 전략 — 고양이 용품 시장

| 카테고리 | 평균 가격대 (네이버 스토어) |
|---|---|
| 일반 낚싯대 | 3,000~8,000원 |
| 프리미엄 낚싯대 (수제) | 10,000~25,000원 |
| 리필 (깃털, 곤충) | 2,000~5,000원 |
| 세트 (낚싯대 + 리필 N개) | 15,000~35,000원 |

아비비 포지션: **프리미엄 수제 + 리필 모델**. "리필 가능한 친환경 수제 장난감"으로 차별화.

## 시즌 캠페인 캘린더

| 시기 | 캠페인 포인트 |
|---|---|
| 1~2월 | 새해 ("올해는 고양이와 더 많이 놀아주세요") |
| 3~5월 | 봄 (활동량 증가) |
| 6~8월 | 여름 (실내 놀이) |
| 9~11월 | 가을 (사냥 본능 시즌) |
| 11~12월 | 블랙프라이데이 + 크리스마스 (선물용) |
