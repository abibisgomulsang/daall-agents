# 네이버광고 브레인 팩 — ROAS 회복 노하우

## 핵심 지표 4개와 진단 룰

| 지표 | 의미 | 좋음 | 나쁨 | 진단 |
|---|---|---|---|---|
| **CTR** | 노출 대비 클릭 | 2~5% | < 1% | 광고 문구/썸네일 문제 |
| **CPC** | 클릭당 비용 | < 500원 | > 1500원 | 입찰가 또는 키워드 |
| **전환율** | 클릭 대비 구매 | 2~5% | < 0.5% | 상세페이지/가격 문제 |
| **ROAS** | 광고비 대비 매출 | > 300% | < 150% | 종합 점검 필요 |

## ROAS < 150% 회복 4단계 룰

ROAS가 낮으면 입찰가부터 만지지 마세요. 순서가 중요합니다.

1. **광고 문구 ↔ 랜딩 페이지 일치도** (60% 영향)
   - 광고에서 "리필 깃털 낚싯대" 강조했는데 랜딩에는 일반 낚싯대 노출되면 이탈
   - → 같은 키워드/같은 이미지로 시작

2. **낭비 키워드 제거** (25% 영향)
   - 클릭은 많은데 전환 0인 키워드 → 즉시 중지
   - CPC × 클릭 ≥ 5,000원인데 매출 0 → 낭비 키워드

3. **입찰가 조정** (10% 영향)
   - ROAS 200% 이상 키워드 → 입찰가 +20% (확장)
   - ROAS 100% 이하 키워드 → 입찰가 -30% (축소)

4. **상세페이지 보완** (5% 영향)
   - 첫 화면에 후킹 + 가격
   - 리뷰 점수/사진 강조

## 낭비 키워드 5대 신호

다음 중 2개 이상 해당하면 **즉시 중지 후보**:

1. 전환율 0% (30일)
2. CPC > 평균 CPC × 2
3. 클릭 후 평균 체류 < 5초
4. 검색어가 우리 상품과 무관 (예: "고양이 사진")
5. 동일 키워드 다른 광고에서도 전환 0

## 광고 문구 작성 룰 (네이버 광고)

- 길이: **15자 이내** (모바일 노출 잘림 방지)
- 가격 표시: 권장 (전환율 +15%)
- 숫자 활용: "3가지 이유", "30일 만에"
- 한정/긴급: "오늘만", "한정 수량" (남용 금지)

좋은 예:
- `고양이 사냥본능 깨우는 고스틱`
- `리필되는 수제 낚싯대 (1+1)`

피하기:
- `최고의 고양이 장난감 정품 판매` (과장)
- `아비비의고물상 공식몰` (브랜드만 강조, 후킹 없음)

## 시간대별 광고 운영 전략

| 시간대 | 노출 효율 | 권장 입찰가 |
|---|---|---|
| 09~12시 | 보통 | 기준 |
| 13~17시 | 낮음 (직장인) | -20% |
| 18~22시 | **가장 높음** (퇴근 후 폰) | +30% |
| 22~24시 | 높음 (집사 잠들기 전) | +20% |
| 00~07시 | 매우 낮음 | -50% 또는 OFF |

## 디바이스별 (모바일 vs PC)

- 모바일: 전환율 더 높음 (고양이 용품은 충동구매 많음)
- PC: 평균 구매 단가 더 높음 (세트 구매 많음)
- → 모바일 입찰가 +20% 기본

## 자주 쓰는 분석 패턴

**일일 점검 (3분):**
1. 어제 ROAS 200% 이상 키워드 → 입찰가 +10%
2. 어제 클릭 5+ 전환 0 키워드 → 중지
3. 일일 광고비 한도 (₩30,000) 70% 초과 여부 확인

**주간 점검 (15분):**
1. 키워드별 ROAS 표 추출
2. 하위 20% 키워드 → 중지 검토
3. 새 키워드 후보 3~5개 추가 (블루오션)

**월간 점검 (1시간):**
1. 전월 대비 ROAS 추이
2. 매출 상위 키워드 톱10
3. 광고 문구 A/B 결과 정리

## 입찰가 변경 시 주의 (실제 적용은 사장님 승인 후)

- 한 번에 ±30% 이상 변경 금지 (오버슈팅 위험)
- 변경 후 **최소 48시간** 데이터 모니터링
- ROAS가 0인 키워드도 일주일은 데이터 모아본 뒤 결정
