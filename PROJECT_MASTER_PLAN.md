# 아비비 AI 회사 구축 마스터 플랜

## 최종 목표

사장님은 결과만 확인하고, AI 직원들이 회의/토론/실행 준비/검수/보고까지 수행하는 AI 회사를 만든다.

## 회사 구조

```txt
사장님
↓
Codex / Hermes / AgentAU / n8n
↓
CEO 오케스트레이터 AI
↓
AI 회의 시스템
↓
전문 AI 직원들
↓
실행 준비물 생성
↓
검수 AI
↓
사장님 승인
↓
실제 실행
```

## 핵심 시스템 6개

1. AI 회의 시스템
2. 광고/마케팅 자동 생성 시스템
3. 스마트스토어 운영 분석 시스템
4. 네이버광고 효율 분석 시스템
5. 고양이 관련 수익성 앱 개발 시스템
6. AI 직원 상태 시각화 시뮬레이터

## 1차 MVP

### AI 회의
주제를 넣으면 여러 AI 직원이 의견을 내고 CEO AI가 결론을 낸다.

### 고스틱 광고 생성
상품 코드 입력 시:
- 광고 문구
- 인스타 피드 문구
- 릴스 대본
- 이미지 편집 지시서
- 해시태그
- 검수 결과
를 생성한다.

### 네이버광고 CSV 분석
CSV 입력 시:
- CTR
- CPC
- 전환율
- ROAS
- 낭비 키워드
- 개선 제안
을 생성한다.

## 2차 목표

- 텔레그램 봇 연결
- n8n 워크플로우 연결
- Ollama 로컬 모델 연결
- 이미지 편집 자동화
- Playwright dry-run 자동화
- AI 직원 사무실 시뮬레이터와 실제 작업 로그 연결

## 3차 목표

- 고양이 장난감 추천 웹앱
- 고양이 MBTI 앱
- 고양이 행동분석 앱
- 스마트스토어/자사몰 연결

## 기술 스택

| 분야 | 기술 |
|---|---|
| 로컬 AI | Ollama |
| 고급 AI | ChatGPT/Claude/Gemini API |
| 오케스트레이션 | n8n, Python |
| 코드 자동화 | Codex |
| 메모리 | PostgreSQL, Redis |
| 브라우저 자동화 | Playwright |
| 앱 개발 | Next.js, FastAPI |
| 이미지 | Stable Diffusion, Canva, PIL |
