# 11.AGENT — 다올에프엔에스 AI 에이전트 팀

주식회사 다올에프엔에스 / 아비비의고물상 브랜드의 AI 에이전트 운영 저장소.

## 에이전트 7명
- **민지** (마케팅매니저) · **하린** (경영전략가) · **현우** (상품기획자) · **도윤** (데이터분석가) · **지호** (해외사업)
- **나린** (네이버 광고 옵티마이저) — 신규
- **세아** (사장님 개인 비서) — 신규

각 에이전트의 상세 역할은 [CLAUDE.md](./CLAUDE.md) 참고.

## 빠른 시작
```bash
# 5인방 셋업
bash abibi-setup.sh

# 나린·세아 추가 셋업
bash narin-seah-setup.sh
```

## 운영 시스템
- openclaw (멀티 LLM 라우팅 + 텔레그램 봇)
- 워크스페이스: `~/.openclaw/ws-{이름}/`
- 설정: `~/.openclaw/openclaw.json`
