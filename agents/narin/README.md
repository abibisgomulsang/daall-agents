# 나린 (Narin)

네이버 광고 운영 전담 에이전트.

## 한 줄 소개
사장님이 시키시는 대로 네이버 광고비를 조절하고, 광고 성과를 분석합니다.

## 주요 기능
- 네이버 검색광고 / 쇼핑광고 성과 모니터링
- 입찰가, 일예산, 캠페인 ON-OFF 자동 조정 (사장님 승인 후)
- CPC 급등 / 전환률 급락 등 이상 알림
- 일·주·월 광고 리포트

## 텔레그램 사용 예시
```
사장님: 고스틱v2 키워드 입찰가 10% 올려줘
나린: 변경 전 평균 CPC 320원 → 변경 후 예상 352원
      예상 일예산 소진: 4시간 빨라질 수 있음
      승인하시겠습니까?
사장님: ㅇㅇ
나린: ✅ 8개 키워드 입찰가 +10% 적용 완료
```
