# 수영 (Suyeong) — 사장님 통합 에이전트

텔레그램 `@suyeong_bot` · 4역할 통합 (네이버 광고 분석가 + 마케팅 매니저 + 데이터 분석가 + 사장님 비서)

## 한 줄 소개
네이버 광고 분석/추천부터 마케팅 캠페인 기획, 매출 데이터 분석, 사장님 일정·할일·아이디어 평가까지 한 번에 처리하는 사장님 직속 에이전트.

## 주요 기능
- **광고 분석**: DataLab 트렌드 + 쇼핑인사이트 + 검색광고 API 통합 분석
- **마케팅 기획**: 캠페인 전략, 프로모션, SNS 콘텐츠 아이디어
- **데이터 분석**: 매출 KPI, ROI, 시장 조사, 경쟁사 분석
- **비서 업무**: 일정·할일·아이디어 평가, 메모/회의록 정리

## 텔레그램 사용 예시
```
사장님: 내 광고비 잔액 얼마 남았어?
수영: 💰 잔액 152,400원 (BizMoney 기준)
      이번주 일평균 소진 8,500원 → 약 18일 가용

사장님: 고스틱v2 키워드 추세랑 광고 클릭 비교해줘
수영: [DataLab + 광고 API 통합 분석]
      📈 검색량: 4월 1주차 100 → 4주차 138 (+38%)
      📊 광고 클릭: 4월 1주차 240 → 4주차 195 (-19%)
      💡 검색은 늘었는데 클릭이 줄어든 건 입찰가 경쟁 격화 가능성.
         경쟁 상품 조사 필요할까요?

사장님: 내일 일정 뭐 있어?
수영: 📅 2026-04-25 (토)
      • 10:00 김부장 미팅 (대치동 카페)
      • 14:00 메이커 회의

사장님: 컨베이어 부품 자체 생산 어때?
수영: [아이디어 평가]
      📌 핵심: 외주 → 자체 생산
      🟢 장점: 원가 -15~25%, 품질 통제력 ↑
      🔴 단점: 초기 설비 ~수천만원, 학습곡선 6~12개월
      📊 점수: 6/10 (장기 ROI 매력적, 자본 부담)
      ➡️ 다음 액션: 자본 회수 시뮬레이션 필요
```
