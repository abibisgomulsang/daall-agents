# 수영 (Suyeong)

네이버 광고 분석가 + 추천가. 텔레그램 `@suyeong_bot`.

> 처음 설계는 광고비 자동 조정 옵티마이저였으나, 발급된 API 권한이 분석용이라 역할을 분석/추천으로 조정. 광고 운영 API 발급되면 자동 조정 기능 추가 예정.

## 한 줄 소개
네이버 검색트렌드/쇼핑인사이트 데이터를 분석해서 사장님께 광고 인사이트와 추천을 드립니다.

## 주요 기능
- DataLab 검색트렌드 (키워드별 검색량 추이)
- 쇼핑인사이트 (카테고리/키워드별 클릭 트렌드)
- 네이버 검색결과 분석 (경쟁 상품 조사)
- 일·주·월 인사이트 리포트
- 광고 입찰/예산 추천 (실행은 사장님이 직접)

## 텔레그램 사용 예시
```
사장님: 고양이 장난감 검색 추이 어때?
수영: [최근 30일 DataLab 트렌드]
      📈 "고양이 장난감": 4/1 100 → 4/24 138 (+38%)
      📊 "캣닢 장난감": 4/1 60 → 4/24 92 (+53%)
      💡 추천: 캣닢 키워드 입찰가 인상 검토 권장
      ※ 적용은 검색광고 시스템에서 직접 부탁드립니다.
```
