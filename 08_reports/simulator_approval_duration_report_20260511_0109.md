# 시뮬레이터 승인 상태 및 작업 시간 연동 보고서

## 1. 수행 목적

AI 사무실 시뮬레이터에서 단순 캐릭터 움직임뿐 아니라 실제 CLI 작업 상태, 승인 파일 현황, 작업별 소요 시간을 볼 수 있게 개선했다.

## 2. 추가 기능

### 승인 상태 표시

시뮬레이터 상태판에 아래 값을 표시한다.

- 승인 대기 수
- 승인 수
- 반려 수
- 최근 승인 요청 파일 5개

현재 피드 기준:

- 대기: 11개
- 승인: 0개
- 반려: 0개

### 작업별 소요 시간 기록

`python -m ai_company.main ...` 명령은 시작과 완료 이벤트를 남긴다.

- 시작: `active`
- 완료: `idle`
- 실패: `blocked`
- 완료/실패 이벤트에는 `duration_seconds`가 기록된다.

예:

```txt
approvals 명령 완료 duration_seconds: 0.011
```

### 시뮬레이터 반영

시뮬레이터가 `activity_feed.js`를 읽어서 아래 내용을 화면에 반영한다.

- 담당 AI 상태
- 최근 CLI 이벤트
- 이벤트별 소요 시간
- 승인 파일 요약
