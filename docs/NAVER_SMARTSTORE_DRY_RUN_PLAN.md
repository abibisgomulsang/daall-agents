# 스마트스토어 / 네이버광고 dry-run 자동화 계획

## 원칙

초기에는 실제 사이트 변경을 하지 않는다.

## 1단계: CSV 분석

- 네이버광고 리포트 CSV 다운로드
- `python -m ai_company.main analyze-ads --csv ...` 실행
- 개선안 생성

## 2단계: 승인 파일 생성

- 변경 후보 키워드
- 변경 전/후 입찰가
- 예상 효과
- 위험성
- 되돌리는 방법

## 3단계: Playwright dry-run

- 로그인 화면 접근까지만
- 변경 버튼 클릭 금지
- 폼 입력 후 제출 전 중단
- 스크린샷 저장

## 4단계: 승인 후 실제 실행

사장님이 승인한 경우에만 실제 반영.
