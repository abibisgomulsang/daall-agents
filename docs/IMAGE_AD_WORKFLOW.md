# 이미지 광고 제작 워크플로우

## 입력

- 상품 이미지
- 상품명
- 상품 특징
- 타겟 고객
- 광고 목적

## 출력

- 1:1 인스타 피드
- 4:5 피드
- 9:16 릴스/스토리
- 스마트스토어 썸네일
- 광고 문구
- 이미지 편집 지시서

## 절차

1. `00_inbox`에 상품 이미지를 넣는다.
2. Codex에게 이미지 파일명을 알려준다.
3. 이미지 광고 AI가 편집 지시서를 만든다.
4. Stable Diffusion/Canva/Photoshop 작업용 프롬프트를 만든다.
5. 결과물을 `03_images`와 `09_approval`에 저장한다.

## 금지 표현

- 치료
- 100% 보장
- 무조건 반응
- 병원/의학적 효과 암시
